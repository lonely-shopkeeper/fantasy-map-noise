fantasy-map-noise
=============

Minimal JavaScript noise generator.

Node.js usage
-------------

```
npm install fantasy-map-generator
```
or
```
yarn add fantasy-map-generator
```

```js
// Generate Noise.
var noise = require('fantasy-map-noise');
noise.perlin2(x, y);
```
